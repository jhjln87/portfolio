Digital Arithmetic Logic Unit

For a college lab, I was to create a four-function arithmetic logic unit with functions XOR, OR, AND, and ADD. I also had to create a multiplexer from scratch and have it determine the function performed. My last task was to write SystemVerilog code to replicate the original schematic. In my free time, I doubled the use of this, and added the functions of LSL, LSR, as well as 1's complement and subtraction. I also combined my four-input multiplexers to make an eight-input multiplexer.

The image contains my SystemVerilog code and an FPGA. I hooked up the FPGA to allow for the two inputs to be the leftmost and rightmost four buttons in the row on the bottom. The five buttons lined up in a plus control my MUX.
